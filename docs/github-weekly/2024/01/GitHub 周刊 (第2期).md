### 2  janhq / jan

```text
🔥 本周 stars：2,681
🔨 语 言：TypeScript
⭐ stars：6,168 
🍴 fork：316 
```

jan 是一个能够完全在本地计算机上运行的 ChatGPT 的开源替代品。用户可以将所需的AI模型下载到本地，做到 100% 离线运行。这样的好处是所有的对话数据和模型输出都在用户的设备上存储，不需要上传到云端服务器，具有较好的安全性和隐私性。当然也可以通过提供 OpenAI 的 key 来使用 GPT 4 模型。
