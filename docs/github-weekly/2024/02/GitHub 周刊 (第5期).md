### 4 facebookresearch / codellama

```text
🔥 本周 stars：1,011

⭐ stars：13,123
```

codellama 是 Facebook 开源的，基于 Llama2 的大语言模型。在开放模型中提供最先进的性能、填充功能、对大型输入上下文的支持以及编程任务的零样本指令跟踪能力。该项目旨在释放大语言模型的力量，让个人、创作者、研究人员和企业可以自由地进行实验、创新和扩展他们的想法。感兴趣的同学可以自行前往下载使用。
